# 福山トレンド

福山市（広島県）の最新トレンド・おすすめ情報を自動収集してお届けするWebアプリ。

## 機能

- **RSS + Google News 自動収集** — 17種類の福山市限定キーワードで絞り込み、広島全域の混入を防止
- **発行元名抽出** — Google News タイトルから実際の新聞社・メディア名を抽出
- **OGP画像取得** — 上位30件の記事サムネイルをサーバーサイドで並列取得
- **フィーチャードカード** — 最注目記事（HOT >= 40）をページ上部にヒーロー表示
- **注目ピックス** — HOTスコア上位5件を横スクロールで表示
- **Claude AI まとめ** — 全体・カテゴリ別のAI要約（タブ切替、日付考慮の詳細プロンプト）
- **トレンドスコア** — 同キーワードが複数記事に出現 = トレンドボーナス付与
- **HOTスコア** — 新着度・サムネイル有無・カテゴリ・トレンドボーナスで算出 🔥
- **NEW バッジ** — 6時間以内の未読記事に緑色の NEW バッジ表示
- **お気に入り** — localStorage永続化、カウントバッジ、/favorites ページ
- **既読管理** — 既読で透明化、StatsBarで件数表示・クリアボタン付き
- **リアルタイム更新時刻** — 「N分前に更新」を1分ごとに自動更新
- **ダークモード** — light/dark/system 3段切替（全コンポーネント対応）
- **タイムライン表示** — 日付グループ化モード（今日/昨日/曜日別）
- **スタガーアニメーション** — カード読み込み時に 50ms ずつズレた入場アニメーション
- **トップローディングバー** — データ取得中は上部に進行バー表示
- **キーボードショートカット** — `/` 検索、`r` 更新、`d` ダーク、`?` ヘルプ
- **自動更新** — 5分ごとサイレント自動リロード + タブアクティブ時
- **PWA対応** — manifest.json、インストールプロンプト、ホーム画面追加
- **モバイルボトムナビ** — ダークモード対応、カウントバッジ付き
- **検索ハイライト** — 大文字小文字を区別しない、特殊文字エスケープ済み
- **共有機能** — Web Share API + クリップボードフォールバック + トースト通知
- **更新トースト** — 手動更新成功/失敗を即座にフィードバック

## セットアップ

```bash
npm install
echo 'ANTHROPIC_API_KEY=sk-ant-xxxx' > .env.local
npm run dev
```

## 環境変数

| 変数名 | 説明 |
|--------|------|
| `ANTHROPIC_API_KEY` | Anthropic APIキー（AI要約に必要、なしでもRSS取得は動作） |

## 技術スタック

- **Next.js 16** (App Router) + **TypeScript** + **Tailwind CSS**
- **Claude API** `claude-sonnet-4-6` — AI要約生成（全体 + カテゴリ別）
- **rss-parser** — RSSフィード取得
- **サーバーサイドキャッシュ** （トレンド: 10分 / AI要約: 20〜30分）
- **OGP スクレイピング** — 上位30件を10件ずつバッチ並列取得

## アーキテクチャ

```
/api/trends         → RSS + Google News → OGP画像 → HOTスコア → キャッシュ
/api/summary        → Claude AI 全体まとめ（30分キャッシュ）
/api/summary/[cat]  → カテゴリ別AI要約（20分キャッシュ）
```

## キーボードショートカット

| キー | 動作 |
|------|------|
| `/` | 検索フォームにフォーカス |
| `Esc` | 検索クリア |
| `r` | データ更新 |
| `d` | ダークモード切替 |
| `?` | ショートカット一覧 |
